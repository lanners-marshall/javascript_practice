Here’s the code for quicksort: def quicksort(array): if len(array) < 2: return array else: pivot = array[0] less = [i for i in array[1:] if i <= pivot] Sub-array of all the elements Base case: arrays with 0 or 1 element are already “sorted.” greater = [i for i in array[1:] if i > pivot] Recursive case less than the pivot Sub-array of all the elements greater than the pivot return quicksort(less) + [pivot] + quicksort(greater) print quicksort([10, 5, 2, 3]) merge sort selection sort breath first search queues enqueue adds item to the q dequeue removes item from the q FIFO data structure: first in, first out think of it as a line of at a store Stack LIFO think of it as like a stack of papers breath first search always finds the shortest path Dijkstra’s algorithm
